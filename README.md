# 스프링부트 입문

인프런 강의에 올라온 김영한 님의 스프링 입문을 통해 만든 회원 관리 예제

### 환경
- 언어 : Java
- IDE  : Intellij
- DB   : H2

### 회원 관리 예제 - 백엔드 개발
- 회원 도메인과 리포지토리 만들기
- 회원 리포지토리 테스트 케이스 작성
- 회원 서비스 개발
- 회원 서비스 테스트

### 스프링 빈과 의존관계
- 컴포넌트 스캔과 자동 의존관계 설정
- 자바 코드로 직접 스프링 빈 등록

### 회원 관리 예제 - 웹 MVC 개발
- 회원 웹 기능 - 홈 화면 추가
- 회원 웹 기능 - 등록
- 회원 웹 기능 - 조회

### 스프링 DB 접근 기술
- H2 데이터베이스 설치
- 순수 JDBC
- 스프링 통합 테스트
- 스프링 JdbcTemplate
- JPA
- 스프링 데이터 JPA

### AOP
- AOP가 필요한 상황
- AOP 적용
